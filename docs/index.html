Sobre o projeto

O keyboard smashing pode ser definido como a entrada ilógica e desordenada de dados, que acaba por comprometer a análise textual por sistemas de software. Dessa maneira, o objetivo do nosso projeto é desenvolver uma ferramenta que possa detectar a ocorrência desse fenômeno, de forma automatizada, fazendo o uso de machine learning.
